## Base Front-end

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Stack

- Next.js
- Typescript
- Tailwind
- Zod
- ReactHookForm
- Shadcn
